# Mood-Based Playlist Generator

## Overview

A simple web app that generates a playlist based on the user's mood input. The app uses predefined playlists for different moods.

## Features

- Enter your mood to receive a playlist.
- Supports moods like "happy", "sad", and "calm".

## Technologies Used

- HTML
- JavaScript

## How to Run

1. Open `playlist_generator.html` in a web browser.

2. Enter your mood in the input box and click "Generate Playlist".
